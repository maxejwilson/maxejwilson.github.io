HTML Structure

The template is based on Bootstrap Framework - http://getbootstrap.com/
The default Bootstrap grid system utilizes 12 columns, making for a 1170px wide depending on your viewport.
Below 480px viewports, the columns become fluid and stack vertically.

Create a .row and add the appropriate number of .col-sm-* columns.

HTML Markup

        <div class="row">
            <div class="col-sm-12">
                Level 1 column
                <div class="row">
                    <div class="col-sm-6">Level 2</div>
                    <div class="col-sm-6">Level 2</div>
                </div>
            </div>
        </div>

If you need more information, please visit this site: http://getbootstrap.com/css/#grid

To change the embedded font, please take a look in the head part of assets/css/theme.css:

Schedule

Schedule tabs created with bootstrap tabs and you can find more documentation here http://getbootstrap.com/javascript/#tabs

Just some notes for easy understanding:

  • <div class="schedule-wrapper clear"> - wrap all schedule content
  • <div class="schedule-tabs lv1"> - lv1 display that tabs first level for days (Day 01, Day 01 .. ) and each tab has id 'tab-first', 'tab-second' ...
  • <div class="tab-content lv1"> - display content for tabs of first level and each tab-pane has correct id 'tab-first', 'tab-second' as tab switcher
  • same for tabs of second level (lv2) it is HALL A, B, ... - '<div class="schedule-tabs lv2">' ... '<div class="tab-content lv2">'
  • all content in tab (example HALL A) wrapped by '<div class="timeline">' and each schedule by '<article class="post-wrap">'

Price Tables

Price tables main wrapper has 'price-tables' class and 3 columns. If you wont 2 or 4 columns you need understanding bootstrap 3 grid system http://getbootstrap.com/css/#grid

Notes for price

  • to markup price with 'Best' add class 'featured' to price table wrapper '<div class="price-table featured">'
  • <h2 class="price-label-title"> - label for the price
  • <span class="price-number"> - price amount
  • <span class="price-unit"> - price currency
  • <div class="price-table-row"> - wrapper for each feature of price

Theme Config

Template has buit-in styles:

  • light / dark version
  • wide / boxed layout
  • ltr / rtl direction
  • color variation

To change this you need need find 'body' tag in html and change/add correct css class
here are default setting example:

             
            <body id="home" class="wide body-light"> ...
             

 

if you wont site dark version just change 'body-light' to 'body-dark'

             
            <body id="home" class="wide body-dark"> ...
             

 

for boxed view change 'wide' class to 'boxed'

             
            <body id="home" class="boxed body-dark"> ...
             

 

same for rtl arabic language just add class 'rtl'

             
            <body id="home" class="boxed body-dark rtl"> ...

Countdown

The countdown is driven by the jQuery Countdown script

You can set you date by open assets/js/theme.js file and change it this code below

            austDay = new Date(austDay.getFullYear() + 1, 1 - 1, 26);

A note on Date - the JavaScript Date constructor expects the year, month, and day as parameters. However, the month ranges from 0 to 11. To make explicit what date is intended (does a month of 3 mean March or April?) I specify the month from 1 to 12 and manually subtract the 1. Thus the following denotes 25 December, 2014. 

            austDay = new Date(2014, 12-1, 25);

For more information check the documentation
